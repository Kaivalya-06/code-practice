A queue is a linear data structure that follows the First-In, First-Out (FIFO) principle, much like a line of people waiting â€“ the first person in line is the first one to be served. It supports two primary operations: enqueue, which adds an element to the rear (end) of the queue, and dequeue, which removes and returns the element from the front (beginning) of the queue. Common operations also include peek or front to view the element at the front without removing it, and isEmpty to check if the queue is empty. Queues are fundamental in managing tasks in operating systems (like print queues), handling requests in servers, and implementing breadth-first search in graph algorithms. Their FIFO nature ensures fair processing of elements in the order they arrive.
